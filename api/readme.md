# kutbi

A way to organise books

## Stack

- Node.js
- Express.js
- MariaDB

### Useful links

- https://www.npmjs.com/package/mysql

### Terminal:

```bash
# Start a Node.js project
npm init -y

# Install dependencies
npm install body-parser
npm install cors
npm install dotenv
npm install express
npm install multer
npm install mysql

## Start the server
npm run start

## Reset the database
npm run dbreset
```
